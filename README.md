# Aurora · Intelligent Creative Co-pilot

Aurora is a marketing experience for an AI creative operating system that helps product teams ideate, prototype, and orchestrate unforgettable launches. It is built with **Next.js 14**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**.

## Tech Stack

- Next.js 14 (App Router)
- React 18 + TypeScript
- Tailwind CSS 3
- Framer Motion

## Getting Started

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to explore the app.

## Scripts

- `npm run dev` – start the dev server with hot reload
- `npm run build` – create a production build
- `npm start` – serve the production build
- `npm run lint` – lint the project with ESLint
- `npm run format` – format sources with Prettier

## Deployment

The project is optimized for Vercel. To deploy manually:

```bash
npm run build
vercel deploy --prod
```
